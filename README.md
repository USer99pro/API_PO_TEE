# API PO TEE Students



## Run Locally

Clone the project

```bash
git clone https://github.com/USer99pro/API_PO_TEE.git
```

Go to the project directory

```bash
cd API_PO_TEE
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```

Shotcut  Install  Dependencies and Start Server
```bash
 npm i && npm run dev
```
